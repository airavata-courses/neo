# neo
Spring 2022 Project

Neo is a microservice architecture based application to visualize Doppler Radar Feed from the NEXRAD system's AWS S3 buckets.

## Technology Stack

- gRPC
- RESTful APIs
- Angular
- TypeScript
- Python
- Flask
- Java
- Nginx
- Docker (with docker-compose)
